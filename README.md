# endpoint-winemaking-process2

## Description

This is a simple API that allows you to manage the winemaking process. It is a RESTful API that allows you to create, read, update and delete the bottling stages of the winemaking process and hosts the production history.

## Acceso a la API

Para acceder a la API, se debe utilizar la siguiente URL: https://my-json-server.typicode.com/SV51-MetaSoft-App-Web/endpoint-winemaking-process2 

## Endpoints

````
 GET /bottling
 GET /productionHistory
````
